# PostNest 📝

PostNest is a simple and modern blog application built with React. Users can explore blog posts, view detailed posts and comments, create new posts, and bookmark posts they want to read later.

## 🚀 Features

- 📚 Browse blog posts
- 📖 View detailed blog posts
- 💬 Read comments on blog posts
- ✍️ Create new blog posts
- 🔖 Bookmark posts
- 🧭 Navigate between pages using React Router
- ⚡ Manage bookmark state using Jotai
- 📱 User-friendly interface

## 🛠️ Technologies Used

- React
- JavaScript
- React Router
- Jotai
- DummyJSON API
- Vite
- CSS

## 📦 Installation

### 1. Clone the repository:
```bash
git clone https://github.com/nebilsefawdin-ops/Blog-app.git
```
### 2. Go to the project directory
```bash
cd blog-app/my-app
```
### 3. Install dependencies:
```bash
npm install
```
## ▶️ Run the project
```bash
npm run dev
```
